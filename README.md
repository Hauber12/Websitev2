# MUKASI Website v2.0 — Modern Alpine

Redesigned website for Musikkapelle Simmerberg e.V.

## What's new in v2.0
- **Fresh typography**: DM Serif Display + DM Sans (distinctive, modern, free)
- **Dark footer**: Professional contrast, better visual hierarchy
- **Cleaner cards**: Border-based instead of heavy shadows, subtle hover lifts
- **Modern CSS**: Scroll-reveal animations, backdrop-filter nav, fluid spacing
- **Lighter weight**: Streamlined CSS (~450 lines vs ~660), optimized JS
- **Better accessibility**: Proper focus states, skip links, ARIA labels
- **Future-proof**: No deprecated features, modern CSS properties
